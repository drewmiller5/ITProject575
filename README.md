# Adaptive Passwordless Authentication System
**IT 575: Advanced Computer Security — Marymount University 2026**

## Overview
This project is a prototype implementation of an adaptive, passwordless 
authentication system built to support research on why passwordless 
authentication is superior to traditional password-based systems.

The system demonstrates that passwordless authentication is:
- **Faster** — no password to type, remember, or reset
- **Simpler** — registration takes seconds with no password complexity rules
- **More effective** — adaptive risk engine detects suspicious behavior 
  that static passwords cannot
- **Auditable** — every login attempt is scored, logged, and traceable
